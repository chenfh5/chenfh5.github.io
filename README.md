# Personal Blog
[chenfh5 blog](https://chenfh5.github.io)

# Quickstart
1. fork [blog project](https://github.com/chenfh5/chenfh5.github.io)
1. rename
1. install [jekyll](https://jekyllrb.com/docs/)
   - [pre install](https://jekyllrb.com/docs/installation/macos/)
1. cd to project's root
1. run `bundle install`
1. modify
   - change configuration to fit your style
   - add new article into `_post` directory
1. bundle exec jekyll serve
    - change directory(cd) to project root
    - run `bundle exec jekyll serve --incremental`
1. push into master after confirmed

# Useful CMD
1. count `post key` from `ls _posts/* | grep "md" | wc -l`, first beginning is zero
