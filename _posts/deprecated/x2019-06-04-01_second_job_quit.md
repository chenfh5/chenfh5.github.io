# Overview
记录一下第二份工作的体验和感觉。总体来说，第二份与第一份差不多。

下面所讲的都是我的个人视角，面向的只是我个人的工作内容，仅代表我自己的拙见。

# 与第一份对比
跟第一份工作比起来，腾讯的文化氛围比京东好，但是我们模块所用的技术没有在京东好。当然这个主要看工作内容和所在组的目标定位。

## 优点
腾讯的优点很多，我说一下我觉得有趣的点，
- 整个大楼是一体的，里面有很多方便的设施，让你更方便的两点一线
    - 可以洗澡
    - 可以健身
    - 可以餐饮
    - 可以理发
    - 可以简单就医
    - 活动空间很大，特别是滨海
    - 有图书角，有电话间，有移动工位
    - 行政很厉害
- 言论相对自由
    - 内部论坛可以匿名谈论许多事情，比如说考核争议，行政吐槽等，虽然最后有些会不了了之，但这是一个发声口
    - `高压线`虽然被调侃为`变压器`，但是大家对其还是很敬畏
- 前沿分享
    - 有很多名人和流量先锋的分享
    - 有很多课程可以在工作时间报名，比如通用素质类，比如技术类
    - 内网上有很多knowledge的分享
- 其他
    - 公仔收集，平时可以参加活动，赢取qq公仔，游戏手办等
    - 很多爱好协会社团可以参加

## 缺点
下面说一下我比较难受的一些点，
- 技术问题，感觉一直在开发和封装API
- 汇报线问题，很长

    `腾讯：我 -> 核心骨干 -> 副组长 -> 组长 -> 副总监 -> 总经理 -> 体系负责人 -> COO`
    
    `京东：我 -> 组长 -> 总监 -> 副总裁 -> CMO`
    
    最让我难受的是我们模块的核心骨干，ta在这一年内没有让我信服，让我觉得ta引领不了我们，
    
    首先代码没写是确定的，从gitlab就能看到。架构能力也没有体现出来，当前模块框架(或者说没有框架，只是API feature的拼凑)也不是ta定的
    
    其次，回到ta说的ta承担了更多的管理工作，我觉得也没有很好地管理到我们模块，工作方向定下之后，后续也一直变来变去，管理工具一开始定义的6大功能，后续完成了几个呢？
    
    离开工位95%不会跟模块内说，我们被他人问到也只能哑口无言。
    
    我觉得ta可能在写代码方便更有优势，为什么非要做自己不擅长的管理。最后导致模块的四分五裂？
    
    考核的时候，组里给了我们模块一个低星名额，我们模块把这个名额给了我，说我的performance不及我的level，所以差评。当我收到这个消息的时候，我心里是很shock的，我自问做得不会是模块性价比最低的人。But whatever既然与直线汇报和组里的文化不match，那么我也下了辞职的决心，春节回来第一天就跟副组长定了辞职意向。
    
    如果在后续工作中，还遇上这样的情况，早点活水或者辞职，别死撑一年，浪费时间。

- 工作环境嘈杂
    
    一天到晚都是张口就来，也不轻声细语，工位没得安静的一刻，要必备降噪耳机。

- 画饼
    
    副组长及以上都说发展很厉害，但从年终bonus的分配就能看到，实际与所说的不符。
    
# 总结
主要是自己心态不稳定，深圳匆匆来也，匆匆去也。心态不定，工作激情就不会饱满，瞻前顾后，顾左右而言他，不能一心一意做事。

那么什么时候才找到自己的锚定点呢？
