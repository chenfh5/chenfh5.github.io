# 常用插件
- .ignore
- BashSupport
- ~~Markdown Navigator~~(table shows perfect，但是比较卡)
- Markdown support(table can not show,但是官方默认)
- String Manipulation
- Maven Helper
- Python
- Scala
- Pig
- ~~CodeGlance~~
- ~~Rainbow Brackets~~
- [GoToTabs](https://stackoverflow.com/a/30237194/9108627)，方便类似XShell, iTerm2那样可以通过`cmd+num`来快速切换tab
    ![image.png](https://upload-images.jianshu.io/upload_images/2189341-3afc4c7b9061b8c0.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

    ![image.png](https://upload-images.jianshu.io/upload_images/2189341-b84492b4907bb00d.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)
    > 另外由于默认的`cmd+1和cmd+7`比较有用，将其置为`alt+1和alt+7`
    
    另外剔除`previous/next tab`防止与`back cmd+[/]`混淆，因此tab跳转全靠`cmd+num`
    
    ![image.png](https://upload-images.jianshu.io/upload_images/2189341-ac07952296c9d05c.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)
    > Navigate的Implementation也改为`cmd+b`，具体实现比声明更常用
